# Recepten app

Made by Joey en Maikel de Haas

## Installing is easy
go to the folder. 

``` bash 
cd server && yarn install 
cd web && yarn install
```

## Useful commands

### Server

``` bash
- yarn watch
- yarn dev
```

### web

``` bash
- yarn watch
- yarn gen
```

Yarn gen is there to generate the graphql file in the 
``` bash
web/src/graphql 
```
